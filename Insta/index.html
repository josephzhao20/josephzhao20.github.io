📥 How to export your Instagram data

  1. Go to Account Center: Click here to open Instagram Account Center
  2. Create export: Look for "Download your information" and click "Create export"
  3. Export to device: Select "Export to device" (not email)
  4. Select data: Check ONLY "Followers and following" — uncheck everything else
  5. Date range: Select "All time"
  6. Format: Select "JSON"
  7. Media quality: Select "Medium"
  8. Download & extract: Download the file and unzip the folder on your computer
  9. Upload folder here: Go to the "Status" tab and upload the main unzipped folder (it will have a name like "instagram-_yourusername_-2025-XX-XX-XXXXX") - don't navigate into subfolders, just upload the main folder
⏱️ Important: Instagram may take a few minutes to prepare your export. If you don't see the download button immediately, check back in a few minutes. When uploading, select the main folder that was created when you unzipped the file - the tool will automatically find the data inside.
